# 大盘云图重构计划

## 目标
高度还原 https://52etf.site/ 的功能与视觉效果

## 当前问题
1. 仅90只股票 → 需要全市场5500+只
2. 无行业板块分组
3. 缺少完整布局（左侧边栏、顶部指数栏、底部筛选）
4. 中文乱码问题
5. 30秒刷新 → 需要8秒

## 新架构设计

### 布局结构
```
┌─────────────────────────────────────────────────────────────┐
│  顶部栏: Logo | 上证指数 深证成指 创业板指 | 时间 | 收藏      │
├──────────┬──────────────────────────────────────────────┤
│          │                                                  │
│  左侧    │              热力图主区域                       │
│  边栏    │          (按行业分组的treemap)                  │
│          │                                                  │
│  - 市场  │                                                  │
│    统计  ├──────────────────────────────────────────────┤
│  - 板块  │  底部栏: [-4%] [-3%]... [0%] ... [+3%] [+4%]   │
│    筛选  │          [09:30] [10:00] ... 复盘按钮          │
│  - 指数  │                                                  │
│    列表  │                                                  │
└──────────┴──────────────────────────────────────────────────┘
```

### 数据流
```
1. 启动时获取全市场股票列表 (5500+ 只，分批)
2. 按行业分组，计算每个行业的总市值
3. 定时8秒刷新一次价格数据
4. 根据涨跌幅计算颜色深浅
5. 根据流通市值计算色块大小
6. 渲染treemap热力图
```

### 行业分类
- 金融：银行、保险、证券
- 消费：白酒、食品饮料、家电
- 医药：制药、医疗器械、医疗服务
- 科技：半导体、电子、通信、软件
- 新能源：光伏、锂电池、风电
- 制造：汽车、机械、化工
- 周期：有色、钢铁、建材
- 基建：房地产、建筑、港口
- 传媒：传媒、游戏、电商

## 实施步骤
1. 更新股票代码数据（全市场5500+只）
2. 重写API模块（支持分批获取、GBK解码）
3. 创建新的布局组件（侧边栏、顶部栏、底部栏）
4. 重写热力图组件（支持行业分组、市值大小）
5. 添加筛选和交互功能
6. 优化样式和动画
7. 测试和部署
